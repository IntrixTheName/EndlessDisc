# Notes

Any additional information about the project that I'm using ofr personal notes is stored here.

## Database - Tags Needed

> For each of the tracks, the following pieces of information will be needed. Note the names used here reflect the information's user-facing intention rather than an ID3v2 tag. This list is meant to detail the information held in tags and is not a complete representation of information stored in the database

- Song/Track Level
  - Title
  - Track.Artist
  - Year
  - Track#
  - Disc#
  - Grouping
  - Comment
  - Owner/Uploader
  - Language
  - Track.Genre
  - Last Modified
- Album Level
  - Album.Name
  - Album.Artist
  - Album.Genre
  - Image ID Link
- Super-Album Level
  - Super-Album.Name
